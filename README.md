##  Overview

This project is a forensic-style financial analysis of the Rothschild family.

It does not aim to speculate.

It audits.

---

##  Objective

To distinguish between:

* Historical influence vs modern relevance
* Personal wealth vs managed capital
* Verified data vs widely circulated claims

---

##  Key Finding

> The widely cited "$350B Rothschild wealth" figure is not personal wealth.
> It represents **Assets Under Management (AUM)** — capital owned by clients.

---

##  Structure

* Phase I → Frankfurt origins
* Phase II → Sovereign financial dominance
* Phase III → Fragmentation and decline
* Phase IV → Modern financial structure
* Wealth Audit → Net worth vs AUM
* Deposition → Myth vs reality

---

##  Core Conclusion

* Estimated combined family net worth: **$20B – $25B**
* Modern influence: **Advisory, not systemic control**
* Narrative scale > actual financial scale
